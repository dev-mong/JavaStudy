< Stack 클래스 >
: 데이터를 쌓아 올리는 형태, 마지막에 저장 한 데이터 부터 데이터를 꺼내옴  (LIFO - 후입 선출 방식)

- 메소드종류
push : 새로운 데이터를 삽입 , top이 1 증가 됨
pop : 데이터를 삭제, top이 1 감소 됨
peak: 데이터를 읽음(마지막 저장 된 데이터 출력), top 변화 없음


< Queue 인터페이스 >
: 데이터를 쌓아 올리는 형태, 처음 저장 한 데이터 부터 데이터를 꺼내옴 (FIFO - 선입 선출 방식)

-메소드 종류
add : 데이터를 삽입
peak : 맨 처음 넣은 데이터를 반환한다
poll : 맨 처음 넣은 데이터를 가리키고 삭제함
